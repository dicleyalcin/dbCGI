dbCGI Documentation

dbCGI provides embedded gene ID mapper REST API mygene and MYSQL to query/retrieve CpG island annotation data.

Currently available organisms and assemblies are as follows:


Organism Assembly
Homo sapiens Dec. 2013 (hg38, GRCh38)
Mus musculus Dec. 2011 (mm10, GRCm38)
Drosophila melanogaster Apr. 2006 (BDGP R5/dm3)
Caenorhabditis elegans Oct. 2010 (Wormbase v.WS220/ce10)
Rattus norvegicus Jul. 2014 (RGSC Rnor_6.0/rn6)


dbCGI uses the following three algorithms to detect CGIs in the genome. dbCGI output files contain results separately listed based on each of the three algorithms.The table below shows the criteria used to define CGIs by the three detection algorithms.


Algorithm G+C% Length Obs/Exp ratio
Gardiner-Garden and Frommer >50% >= 200bp >= 0.60
Takai-Jones >55% >= 500bp >= 0.65
Ponger-Mouchiroud >50% >= 500bp >= 0.60

Annotation Strategy

Input File

dbCGI accepts plain text files as input. There are two different input file types:
    Gene ID File: In this file each line should contain one gene identifier with no separators like ,/;/:/ etc. The embedded ID mapper accommodates a wide range of gene identifier types. Here you can find a sample Gene ID File. Chromosomal Location File: In this file each line represents one continuous interval on the genome. Chromosome number is indicated by ChrN (e.g. Chr1, Chr2, etc.) followed by the start and end positions, all separated by '-' (e.g. Chr3-736828-1364826). If you want to query the entire chromosome, do not enter any position information following the chromosome number. X and Y chromosomes are denoted by ChrX and ChrY, respectively. Here you can find a sample Chromosomal Location File.





Please make sure you choose the organism and assembly before you upload your file and submit your query.


The output

The output file contains an annotated list of CGIs. CGI names represent the detection algorithm, chromosome number, and rank; e.g. "TJ_Chr7_CGI 874" represents the 874th overall CGI found in the genome that lies on Chromosome 7, according to the Takai-Jones detection algorithm.

For each CGI, the output file lists start and end positions, G+C%, Length, and observed/expected GC ratio. The output files show slight difference in their formats based on the input file type being Gene ID File ( See File ) or Chromosomal Location File ( See File ).
