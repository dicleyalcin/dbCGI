dbCGI is a web application for analyzing and annotating CpG islands (CGIs) based on their locations, sequence properties, and associations with genes. dbCGI has two main query objectives:
  1. Given a list of genes, the user can retrieve annotated CGIs associated with the genes, based on the CGIsâ€™ locational proximity.
  2. Given a list of chromosomal locations, the user can retrieve annotated CGIs and associated genes within the given range.


dbCGI currently employs three CpG island detection algorithms and is available for five organisms. Please refer to the tutorial for more details. dbCGI annotates CGIs in relation to genes using the following criteria:

CGI Types

Type Definition
Promoter CGI overlaps with [-5kb,TSS] (TSS: Transcription Start Site)
Intragenic CGI overlaps with [TSS,TES] (TES: Transcription End Site)
Gene-Terminal CGI overlaps with [CDSEnd, TES] (CDSEnd: Coding Sequence End Site)
Intergenic CGI is not associated with any gene
